# Vestra Protocol (Public)

Welcome to the public docs for Vestra (formerly CRDT). This site mirrors the open protocol materials used by the community: overview, litepaper, full whitepaper, risk models, and FAQ. No wallet or admin access required.

## How to read

- Start with the **Overview** for the problem, who it serves, and where to go next.
- Skim the **Litepaper** for the concise story.
- Dive into the **Whitepaper** for full protocol, valuation, and enforcement detail.
- Check **Risk Models** for LTV math and simulations.
- Use **FAQ** for common answers.

## Running locally (Mintlify)

1) Install the Mintlify CLI (one time): `npm i -g mint`  
2) From `/UNLOCKD/mintlify`: `mint dev`  
3) Open http://localhost:3000 to preview.
