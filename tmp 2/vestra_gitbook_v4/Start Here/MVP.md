# MVP: The Internal Command Center Scope

The Vestra MVP is designed to prove the core primitive on the **Sepolia Testnet** before scaling to high-liquidity chains.

## MVP Objectives
1.  **Core Valuation Engine**: Implementation of the `dDPV` formula for simulated ERC-20 vesting streams.
2.  **Loan Lifecycle**: End-to-end flow from loan creation (stablecoin payout) to repayment or auction.
3.  **Privacy Hook**: Successful integration of the Relayer-driven stealth address payout.
4.  **Omega v0.1**: Predictive monitoring of mock multi-sig dilution events.

## Build milestones
- **Stage 1**: Smart Contract core (Valuation, Vault, LoanManager).
- **Stage 2**: Relayer & Backend Agent (Pricing feeds, Stealth Address derivation).
- **Stage 3**: Command Center UI (Borrow/Lend Dashboards).
- **Stage 4**: Staged Auction Simulator.

## Target Metrics for Piloting
- **10,000 Testnet Users**: Via incentivize airdrop points.
- **$100M+ Simulated TVL**: To test the robustness of the Staged Auction under load.
