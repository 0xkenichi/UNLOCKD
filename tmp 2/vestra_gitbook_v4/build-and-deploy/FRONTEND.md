# Frontend: Command Center UI Vision

The Vestra UI is not a "dashboard"—it is a **Command Center**.

## Design Philosophy
- **Institutional Aesthetic**: Deep navy colors, electric cyan accents, and clear, uncluttered data tables.
- **Micro-Animations**: Real-time ticker updates for `dDPV` and `Omega` triggers.
- **Visual Risk Indicators**: "Liquid-gold" callouts for high-risk positions or impending auctions.

## Core Screens
1.  **Sovereign Dashboard**: Overview of all active loans and collateral health.
2.  **Borrower Terminal**: One-click borrowing with clear LTV sliders and privacy toggles.
3.  **Lender Vaults**: APY tracking and risk-exposure management.
4.  **Strategic Auction Hub**: Bidding interface for market makers.

## Tech Stack
- **Framework**: React / Vite.
- **Web3 Connectivity**: Wagmi & RainbowKit.
- **State Management**: Zustand (for real-time pricing sync).
