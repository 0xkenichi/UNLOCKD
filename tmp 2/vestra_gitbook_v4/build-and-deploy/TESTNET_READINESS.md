# Testnet Readiness Criteria

Before moving to Phase 2 (Public Incentivized Testnet), the following criteria must be met:

1.  **Smart Contracts**: 
    - [x] Deployed and verified on Sepolia.
    - [x] Valuation logic matches Python Monte Carlo outputs within 0.1% margin.
2.  **Infrastructure**:
    - [x] Omega Relayer uptime > 99.9% over 7 days.
    - [x] Redis cache successfully handling 100+ requests per second.
3.  **UI/UX**:
    - [x] Stealth payout flow verified on mobile.
    - [x] Error handling for "Insolvent Collateral" clearly displayed.

**Target Launch Date**: April 2026.
