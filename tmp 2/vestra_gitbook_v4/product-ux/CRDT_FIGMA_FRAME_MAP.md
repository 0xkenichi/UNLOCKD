# Figma Frame Map: Command Center UI

Mapping the institutional user experience.

## Frame Index
1.  **Main Navigation Sidebar**: Collapsed by default for a clean "Institutional Terminal" feel.
2.  **Global Metrics Bar**: TVL, Protocol Revenue (24h), and Average Omega Score.
3.  **Borrow Flow Modal**: 4-step wizard (Select Collateral -> Set Amount -> Privacy Opt-in -> Confirm).
4.  **Risk Monitor**: Detailed breakdown of the dDPV inputs for an active loan.
5.  **Governance Terminal**: Gauge voting and $CRDT locking interface.
