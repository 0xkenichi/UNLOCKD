# Glossary

- **DPV**: Discounted present value of locked tokens.
- **LTV**: Loan-to-value ratio used to cap borrowing.
- **Unlock time**: Timestamp when vesting ends and tokens can be released.
- **Settlement**: On-chain enforcement at unlock time.
- **Seize**: Amount of collateral transferred for repayment.
- **Liquidation**: Swap seized tokens for USDC to repay the pool.
- **Collateral ID**: Identifier for a vesting position in the adapter.
