### 1) Protocol Security Engineer (Solidity)

What you do:
- harden lending + settlement + liquidation flows
- close attack surfaces before mainnet
- own test + invariant + incident readiness

You should have:
- shipped audited contracts in production
- deep understanding of token accounting and edge cases
- experience with threat modeling and exploit prevention

### 2) Senior Full-Stack Product Engineer

What you do:
- own Borrow -> Portfolio -> Repay flow
- reduce user friction and improve conversion
- improve backend reliability and observability

You should have:
- strong React + Node backend execution
- wallet/web3 integration experience
- history of shipping high-quality UX fast

### 3) Risk Quant / Protocol Analyst

What you do:
- tune LTV/DPV parameters
- run stress tests and scenario simulations
- translate risk outputs into governance proposals

You should have:
- Monte Carlo/stat modeling background
- DeFi market intuition
- ability to communicate risk simply

### 4) BD / DevRel Operator

What you do:
- source and close pilot DAOs/protocols
- run partner pipeline + LOIs + activation
- build trust narrative in public

You should have:
- web3 partnerships track record
- clear writing + community credibility
- execution bias and weekly cadence discipline
