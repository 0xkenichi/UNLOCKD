# Pilot Launch Checklist

Use this checklist before launching any partner pilot.

## Pre-Launch

- [ ] Partner profile completed (contacts, chain, compliance constraints)
- [ ] Vesting contracts validated against supported adapter paths
- [ ] Pool risk settings approved by risk committee
- [ ] Security checklist reviewed for pilot scope
- [ ] Incident response contacts assigned

## Technical Readiness

- [ ] Contracts deployed and addresses verified
- [ ] Backend health checks passing
- [ ] KPI endpoints reachable (`/api/kpi/dashboard`)
- [ ] Analytics ingestion tested (`/api/analytics`)
- [ ] CSV exports tested (`/api/exports/kpi.csv`)

## Operational Readiness

- [ ] Pilot onboarding guide shared with partner
- [ ] Weekly review cadence scheduled
- [ ] Escalation path documented (Sev 0/1)
- [ ] Treasury / fee routing expectations documented

## Go-Live Gate

- [ ] Dry run completed end-to-end
- [ ] At least one test borrower and one test lender confirmed
- [ ] Monitoring dashboard populated with baseline metrics
- [ ] Public or partner-facing status page prepared

## Post-Launch (Week 1)

- [ ] Publish first KPI snapshot
- [ ] Review borrower funnel drop-offs
- [ ] Review defaults/delinquency signals
- [ ] Capture partner feedback and required product fixes
